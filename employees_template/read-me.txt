Так вот, верстка здесь очень простая. Суть в том, что визуальный макет делится на компоненты и в следствии верстается этот макет такими вот компонентамим, которое в отдельной папке
Каждый компонент независим и имеет даже свой css файл. Конечно можно все стили прописывать в общий css файл, но поверьте намного удобнее делать как здесь
Так вот, в компонентах есть css и js файл, в который мы импортируем css файл, а js файлы импортируем в главынй компонент app.js, а app.js в index.js, который уже отрисовывается на сатйе
Главное не запутаться). Также здесь подключены библиотеки bootstrap и fontawesome cdn ссылками
